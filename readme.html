UniTube installation

System requirements

  • Modern Operating Systems, such as Windows XP, Windows Vista, Windows 2003 Server, Linux (Ubuntu). Although UniTube has no problem to run under Windows, it is highly recommended to use a Linux server, especially a Ubuntu server. All our testings are carried under Ubuntu .
  • Java JDK 5.0 or JDK 6.0
  • Java application server (sucn as Tomcat 5.5, Tomcat 6)
  • FFmpeg
  • OpenOffice
  • SWFTools
  • ImageMagick

Download

To run UniTube on you server, all you need to do is downloading unitube.war from our web site. If you want to customize UniTube to meet your requirement, you can download source code.

Install

UniTube is a standard Java web application. If you have already had Java JDK and Java web application server (such as Tomcat) installed, all you need to do is dropping unitube.war to right place (for Tomcat, it's under ${TOMCAT_HOME}/webapps/). It will be uncompressed automatically.

To access it, just point your browser to http://localhost:8080/unitube/ (assume you are running your Java application server on port 8080). If you have no problem to access it, congratulations, you have installed UniTube web application successfully.

Unfortunately, that's not all (I wish it's as simple as that).

UniTube is using many external open source and free software. To be able to use all function of UniTube, you have to install others software as well.

FFmpeg

We use FFmpeg to do audio and video conversion.

OpenOffice

We use OpenOffice to convert office files to flash or PDF files.

SWFTools

We use SWFTools to get dimension information from flash (.swf) files.

ImageMagick

We use ImageMagick to do image conversion, also we use ImageMagick to convert PDF files to a bunch of images.
